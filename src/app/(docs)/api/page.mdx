# Andromeda API

The Andromeda API is a REST API that gives you full programmatic access to Galacti services. All responses are JSON.

## Base URL

```
https://api.galacti.org/v1
```

## Authentication

Include your API key as a Bearer token in every request:

```bash
curl https://api.galacti.org/v1/ping \
  -H "Authorization: Bearer gal_your_key_here"
```

> API keys can be generated and revoked from **Settings → API Keys** in the [dashboard](https://dashboard.galacti.org). Never commit your key to a repository.

When a rate limit is exceeded, the API returns `429 Too Many Requests` with a `Retry-After` header indicating how many seconds to wait.

## Errors

All errors follow the same shape:

```json
{
  "error": {
    "code": "not_found",
    "message": "The requested resource does not exist.",
    "status": 404
  }
}
```

## SDKs

Official SDKs are in development. In the meantime, any HTTP client works. Examples throughout the docs use `curl` and plain `fetch`.

```ts
const res = await fetch("https://api.galacti.org/v1/ping", {
  headers: {
    Authorization: `Bearer ${process.env.GALACTI_API_KEY}`,
  },
});
const data = await res.json();
```
