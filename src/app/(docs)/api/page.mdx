# Andromeda API

The Andromeda API is a REST API that gives you full programmatic access to Galacti services. All responses are JSON.

## Base URL

```
https://api.galacti.org/v1
```

## Errors

All errors follow the same shape:

```json
{
  "error": {
    "code": 404,
    "message": "The requested resource does not exist",
    "status": "not_found"
  }
}
```
